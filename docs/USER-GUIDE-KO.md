# GonpunClaw PolicyMap 사용 안내

## 1. 접속
- 서비스 주소: <https://gonpunclaw-policymap.vercel.app>
- 샘플 지도: <https://gonpunclaw-policymap.vercel.app/demo>
- 업로드 페이지: <https://gonpunclaw-policymap.vercel.app/upload>

## 2. 무엇을 하는 서비스인가
주소가 들어 있는 엑셀 파일을 업로드하면, 주소를 자동으로 좌표로 변환해 공개 지도를 만들어 주는 서비스입니다.
샘플 지도에서는 파일을 올리지 않아도 검색, 필터, 표 보기를 먼저 체험할 수 있습니다.

활용 예시:
- 정책사업 대상지 위치 시각화
- 지원시설 분포 지도
- 거점기관 위치 공유
- 현장 점검 대상지 맵 만들기

## 3. 권장 파일 형식
- 권장: `.xlsx`
- 가능: `.csv`
- 현재 실사용 기준으로는 `.xlsx`가 더 안정적입니다.

## 4. 엑셀 파일 형식
기본 구조는 아래와 같습니다.

한 행은 지도에 표시될 위치 1개입니다. 예를 들어 시설 10개를 올리려면 헤더 아래에 데이터 행을
10개 넣으면 됩니다. 여러 시트가 있어도 첫 번째 시트만 읽습니다.

예를 들어 아래 한 줄은 지도에서 `예시복지관` 위치 1개로 표시됩니다.

| A열 주소 | B열 이름 | C열 대표값 | D열 분류 | E열 이후 |
| --- | --- | --- | --- | --- |
| 서울 서초구 반포대로 58 | 예시복지관 | 48 | 복지 | 담당부서, 비고 |

| 열 | 의미 | 예시 |
| --- | --- | --- |
| A열 | 주소 | 세종특별자치시 도움6로 11 |
| B열 | 이름 | 정부세종청사 |
| C열 | 대표값 | 100 |
| D열 | 분류 | 행정 |
| E열 이후 | 공개 팝업에 표시될 추가 정보 | 담당부서, 비고 등 |

### 최소 조건
- A열 헤더는 `주소`, `소재지`, `도로명`, `address` 중 하나처럼 주소 컬럼임이 드러나야 합니다.
- 주소가 비어 있는 행은 자동으로 제외될 수 있습니다.

## 5. 업로드 화면에서 입력하는 값
### 지도 제목
예: `세종 공공시설 분포`

### 설명
예: `2026년 기준 주요 공공시설 위치`

### 값 컬럼 라벨
예: `사업비`, `인원`, `면적`

C열 대표값은 위치 1개에 붙일 숫자 1개입니다. 여러 숫자 지표가 필요하면 대표값 하나를 C열에
두고, 나머지는 E열 이후 추가 정보로 넣는 편이 좋습니다.

### 단위
예: `억원`, `명`, `㎡`

### 분류 컬럼 라벨
예: `기관유형`, `사업유형`

## 6. 업로드 후 받게 되는 것
업로드가 성공하면 아래 정보가 생성됩니다.

### 공개 지도 URL
- 누구나 볼 수 있는 링크입니다.
- 외부 공유용으로 사용합니다.

### 관리 페이지 URL
- 업로드한 지도를 수정하거나 삭제할 수 있는 페이지입니다.

### 관리 토큰
- 수정/삭제에 꼭 필요합니다.
- 업로드 직후 안전한 곳에 따로 저장해야 합니다.
- 잃어버리면 관리가 어려워집니다.

## 7. 수정과 삭제
관리 페이지(`/manage/[slug]`)에서 아래 작업이 가능합니다.
- 제목 수정
- 설명 수정
- 값/단위/분류 라벨 수정
- 공개 여부 변경
- 지도 삭제

## 8. 주의사항
- 주소는 가능한 한 도로명 기준으로 정확하게 쓰는 편이 좋습니다.
- 너무 모호한 주소는 좌표 변환이 실패할 수 있습니다.
- 현재 운영 기준으로는 `.csv`보다 `.xlsx` 업로드를 권장합니다.
- 업로드한 내용은 공개 지도에 표시되므로 개인정보나 민감정보가 들어 있는 열은 제거한 뒤 올려야 합니다.
- 관리 토큰은 반드시 별도 보관해야 합니다.

## 9. 가장 쉬운 사용 순서
1. 엑셀을 `.xlsx`로 저장
2. `/upload` 접속
3. 제목, 설명 입력
4. 파일 업로드
5. 공개 링크와 관리 토큰 저장
6. 공개 링크를 공유

## 10. 추천 첫 테스트
먼저 샘플 지도에서 결과 화면을 확인하고, 이후 2~5행짜리 작은 파일로 업로드를 테스트한 뒤
실제 대량 파일을 올리는 것을 권장합니다.
